HOW TO RUN:
1. Open terminal in the project folder
2. Install dependencies: pip install -r requirements.txt
3. Download NLTK VADER lexicon (first run will do it automatically)
4. Run: streamlit run app.py
5. The dashboard opens in your browser (usually http://localhost:8501)

HOW TO USE (Quick Guide):
- Use the top menu to navigate between four pillars:
  GLOBAL MACRO → economic indicators & causality
  2021 STOCK CASE STUDY → sentiment timelines, anomalies, comparisons
  AI ENGINE & PATTERN MINING → model comparison, topic mining, patterns
  KNOWLEDGE & THEORY → Learning Hub (quizzes), Glossary, Methodology
- Sidebar filters: date range, company, sentiment, min score. Some extra filters appear per page.
- Hover over any chart for details.
- Dark/light mode toggle in sidebar.
- Onboarding tour (first visit) guides you through key features.

TROUBLESHOOTING:
- If FRED API fails, the economic dashboard will show limited data – fallback to cached data.
- If models are missing, they will be re‑trained automatically.
